# Commands

## Command Mode

When Vim mode is enabled and you are in **Normal Mode**, you can type `:` to enter **Command Mode**. Once in Command Mode, type the command and press `Enter` to execute it.

| Command | Action |
|---------|--------|
| `:quit` / `:q` | Quit the application |
| `:debug <message>` | Print a debug message to the logs |
| `:logs` | Open logs panel |
| `:status <online\|dnd\|idle\|invisible\|offline> <text>` | Change user status & custom status |
